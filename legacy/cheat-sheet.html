RADICAL SQL TIPS

SELECT - data type/column (ex. state_name)
FROM - table name (ex. regions)
INNER JOIN - other table (ex. states)
ON - continuing the join (ex. regions.id = states.region_id)
WHERE - conditional
BETWEEN x AND y, <, >,
ORDER BY - order data (ex. region_id DESC) DESC, ASC UPDATE - updates data in a database DELETE - deletes data from a database INSERT INTO - inserts new data into a database CREATE DATABASE - creates a new database ALTER DATABASE - modifies a database CREATE TABLE - creates a new table ALTER TABLE - modifies a table DROP TABLE - deletes a table CREATE INDEX - creates an index (search key) DROP INDEX - deletes an index
